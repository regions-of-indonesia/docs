## Installation

```bash
npm install @regions-of-indonesia/client
# or
yarn add @regions-of-indonesia/client
# or
pnpm add @regions-of-indonesia/client
```

## Usage

### Basic

```typescript
import { create } from "@regions-of-indonesia/client";

const client = create();

client.province.find(); // Promise<Region[]>
client.province.find.by("11"); // Promise<Region>
client.province.search("a"); // Promise<Region[]>

client.district.find("11"); // Promise<Region[]>
client.district.find.by("11.01"); // Promise<Region>
client.district.search("a"); // Promise<Region[]>

client.subdistrict.find("11.01"); // Promise<Region[]>
client.subdistrict.find.by("11.01.01"); // Promise<Region>
client.subdistrict.search("a"); // Promise<Region[]>

client.village.find("11.01.01"); // Promise<Region[]>
client.village.find.by("11.01.01.2001"); // Promise<Region>
client.village.search("a"); // Promise<Region[]>

client.region("11"); // Promise<Region>
client.region("11.01"); // Promise<Region>
client.region("11.01.01"); // Promise<Region>
client.region("11.01.01.2001"); // Promise<Region>

client.search("a"); // Promise<Region[]>
client.search.provinces("a"); // Promise<Region[]>
client.search.districts("a"); // Promise<Region[]>
client.search.subdistricts("a"); // Promise<Region[]>
client.search.villages("a"); // Promise<Region[]>
```

Custom baseURL (change to your self-hosted services)

```typescript
import { create } from "@regions-of-indonesia/client";

const client = create({
  baseURL: {
    dynamic: "YOUR_DYNAMIC_BASE_URL", // default : "https://regions-of-indonesia.deno.dev"
    static: "YOUR_STATIC_BASE_URL", // default : "https://regions-of-indonesia.github.io/static"
  },
});
```

Disable static fetcher

```typescript
import { create } from "@regions-of-indonesia/client";

const client = create({
  static: false, // default : true
});
```

Disable logger

```typescript
import { create } from "@regions-of-indonesia/client";

const client = create({
  logger: false, // default : true
});
```

With middleware (useful for testing)

```typescript
import { create, delay, cache } from "@regions-of-indonesia/client";

const client = create({
  middlewares: [delay(), cache()], // default : []
});
```
